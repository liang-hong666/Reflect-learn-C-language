## 题目
给定3个整数a,b,c，计算表达式 (a+b)*c 的值。
## 输入格式
输入仅一行，包括三个整数 a,b,c，数与数之间以一个空格分开。
## 输出格式
输出一行，即表达式的值。

#### 解法一
```C
#include<stdio.h>
int main()
{
    int a,b,c;
    scanf("%d %d %d",&a,&b,&c);
    printf("%d\n",(a+b)*c);
    return 0;
}
```
编译迅速，不需要调用函数。

#### 解法二
```C
#include<stdio.h>
 int calculate(int x, int y, int z)
{
    int result;
    result = (x+y)*z;
    return result;
}
int main()
{ 
    int a, b, c; 
    int s;
    scanf("%d %d %d",&a,&b,&c);
    s = calculate(a,b,c);
    printf("%d\n",s);
    return 0;
}
```
调用函数，calculate函数还可以在其他函数中使用。

## 题目
输入两个整数，输出最大值。
例如：输入：1，2 输出：2

#### 解法一
```C
#include<stdio.h>
int main()
{
    int a,b;
    printf("please input two number:");
    scanf("%d %d",&a,&b);
   
    if(a>b)
    {
        printf("最大值=%d\n",a);
    }
    else
        printf("最大值=%d\n",b);
    return 0;
}
```
虽然直观，但是不可以扩展，即max值不能在后面继续使用。
#### 解法二
```C
int main()
{
    int a,b;
    printf("please input two number:");
    scanf("%d %d",&a,&b);
    int max = 0;
    if(a>b)
    {
        max = a;
    }
    else
        max = b;
    printf("最大值=%d",max);
    return 0;
}
```
可以扩展，且可以比较多个数的大小
#### 解法三
```C
int main()
{
    int a,b;
    printf("please input two number:");
    scanf("%d %d",&a,&b);
    int max = b;
    if(a>b)
    {
        max = a;
    }
    printf("最大值=%d",max);
    return 0;
}
```
可以扩展，且可以比较多个数的大小，相比解法二更简洁。
思路：先假设多个数中的最大值，然后依次比较并且更新。
#### 解法四
```C
int main()
{
    int a,b,max;
    printf("please input two number:");
    scanf("%d %d",&a,&b);
    max = (a>b)?a:b;
    printf("最大值为%d",max);
    return 0;
}
```
可以扩展，比较两个数大小最简洁方式。
